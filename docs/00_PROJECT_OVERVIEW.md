# Project Overview

This repo holds the planning + architecture for the HexForge Portable Lab Assistant (PLA).

Scope (initial):
- Controlled sandbox VMs (Windows + Linux) for testing
- “Eyes”: camera / screen capture for observation
- “Hands”: HID test rig for keyboard/mouse *in a lab-safe, authorized environment only*
- Optional: small projector + speaker for portability
- Integration point: HexForge Lab Assistant (web) as the control UI

Non-goals:
- Anything designed for unauthorized access or covert interaction
