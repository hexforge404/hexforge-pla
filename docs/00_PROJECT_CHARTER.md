## Goal
Build a portable internal lab assistant that can observe screens via camera/capture and execute *approved* input actions via HID.

## Non-Goals
- No autonomous control
- No use on systems without explicit owner consent
- No persistence or stealth behavior

## Success Criteria (MVP)
- Camera feed captured and viewable
- Assistant summarizes current screen state
- Assistant proposes next steps
- User approves/denies actions
- Session logging enabled
